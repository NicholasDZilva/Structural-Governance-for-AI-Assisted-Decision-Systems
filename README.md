# Structural Governance for AI-Assisted Decision Systems

**File Hash:** `9eabcdff1a7e560f74424d9bfd2f9d6f6a1c9b1eeb1c884988d0aeb5c9db83e8`

This repository contains a public research paper, `StructuralGovernance.txt`, exploring **structural governance** and **framing bias** in AI-assisted decision systems. The paper focuses on:

- How problem framings can systematically exclude classes of solutions.
- The conceptual distinction between institutional and peripheral solution frames.
- High-level audit approaches to detect structural omissions in solution spaces.

> **Note:** This paper is **conceptual**. It does **not include proprietary methods, code, or operational frameworks** developed at Invariant Systems. The focus is on public, shareable ideas and the documented problem space.

## Purpose

This work is intended to:

- Share insights on AI framing bias and structural governance.
- Encourage discussion and critique on problem-framing effects in AI-assisted decision-making.
- Provide a citable reference for conceptual research without exposing proprietary tools.
